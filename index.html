BBQr Maker - ColdCard Q & Wallet Compatibility Bridge Tool

Bridge Tool for Wallet Compatibility

BBQr Maker serves as a crucial bridge between ColdCard Q hardware wallets and wallet software that doesn't natively support the BBQr protocol. This tool solves the compatibility gap, enabling users to benefit from modern hardware wallet security while continuing to use their preferred wallet software.

Primary Use Cases

  • Enable ColdCard Q integration with wallets lacking BBQr support
  • Bridge older wallet software with modern hardware wallet capabilities
  • Provide animated QR workflow for wallets with basic QR scanning
  • Facilitate secure air-gapped transaction signing across different wallet ecosystems

Features and Capabilities

  • Convert Bitcoin Partially Signed Bitcoin Transactions (PSBT) to animated BBQr codes
  • Bridge compatibility between ColdCard Q and non-BBQr supporting wallets
  • Enable wallet software ecosystem integration without native BBQr support
  • Completely offline operation - no internet connection required after initial load
  • Zero data transmission - all processing happens locally in your browser
  • Support for large PSBT files with automatic frame optimization
  • Hardware wallet compatible QR codes for secure transaction signing
  • ColdCard inspired retro LCD interface design
  • Progressive Web App (PWA) - install for offline use
  • Animated QR display with adjustable timing for easy scanning
  • Advanced BBQr protocol implementation with multi-frame support
  • Automatic fallback strategies for complex transactions

Security and Privacy

BBQr Maker prioritizes your security and privacy:

  • No data is ever sent to external servers
  • All PSBT processing happens locally in your browser
  • No tracking or analytics
  • Open source and transparent
  • Works completely offline once installed

How to Use

  1. Paste your Bitcoin PSBT (Base64 format) into the text area
  2. Click "Generate BBQr" to create animated QR codes
  3. Use your hardware wallet or compatible software to scan the animated QR sequence
  4. Complete your Bitcoin transaction signing securely offline

Wallet Compatibility & Integration

BBQr Maker specifically bridges the gap between:

  • Hardware Wallets: ColdCard Q, ColdCard Mk4, and other BBQr-capable devices
  • Wallet Software requiring BBQr bridge: Bitcoin Core, Electrum (older versions), Blue Wallet, Green Wallet, and other wallets without native BBQr support
  • Native BBQr Wallets: Sparrow Wallet, Specter Desktop, and other modern Bitcoin security tools

Workflow Integration

Typical usage scenario: Your wallet software creates a PSBT → BBQr Maker converts it to animated QR → ColdCard Q scans and signs → Return signed transaction to your wallet software.
